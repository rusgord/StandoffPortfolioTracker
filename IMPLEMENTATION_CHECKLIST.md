## Базовые данные

### Используемые сущности:
- ✅ **ItemBase** - базовый предмет
- ✅ **MarketHistory** - история цен
- ✅ **GameCollection** - коллекции предметов
- ✅ **ItemRarity** - типы редкости
- ✅ **ItemType** - типы предметов

### SQL запросы:
```sql
-- История цен за последние 90 дней
SELECT mh.RecordedAt, mh.Price 
FROM MarketHistory mh 
WHERE mh.ItemBaseId = @itemId 
AND mh.RecordedAt >= DATEADD(day, -90, GETUTCDATE())
ORDER BY mh.RecordedAt ASC
```

## Примечания для разработчиков

1. **Chart.js** - используется версия 4.4.0 от CDN
2. **Данные цен** - требуется, чтобы в БД были записи в таблице `MarketHistory`
3. **Пример использования**: `await ItemService.GetPriceHistoryAsync(itemId, 90)`
4. **Дизайн** - используется палитра фиолетово-голубого градиента
5. **Рецепт обновления цен** - вызывать `PriceParserService.UpdateAllPricesAsync()`
